# cookBook
An e-recipes book 
#Project Overview
 1.A web application for managing recipes.
 2.Allows users to add, edit, view, and delete recipes.
 3.Supports ingredient-based search and categorization.
 4.Includes user login and personalized meal planning.
 
#Project Objectives
1.Build a modular Java web application using Servlets & JSP.
2.Demonstrate JDBC CRUD with a normalized database.
3.Implement user authentication & session management.
4.Provide recipe and ingredient management features.
5.Add meal planning and basic reporting for usability.
6.Follow MVC architecture and Java OOP principles.

#Proposed Workflow:
1.User logs into the system
2.User views list of recipes
3.User adds/edits recipes
4.User searches recipes by ingredients/cuisine
5.User adds recipes to favoritesor meal plan
6.User exports shopping list (CSV)
7.System runs a background suggestion thread 

#Technologies & Tools
 1.Backend:Java, Servlets, JSP,JDBC
 2.Database:MySQL 
 3.Frontend:HTML, CSS, Bootstra
